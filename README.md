# Plant Picker

A browser-based tool for researching and comparing ornamental flowers from Russell's Garden Center.

**Zone:** 6b/7a (Boston area) · **Goal:** Continuous spring-to-fall blooms

## Features

- Filter by bloom season, sun, water, color, zone, and more
- Mark favorites and compare up to 3 flowers side by side
- Data sourced from Russell's Garden Center

## Usage

Open index.html in a browser to start exploring flowers.

## Data Sources

- Russell's Garden Center (local nursery, Boston area)
