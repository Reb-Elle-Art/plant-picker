### Specific DB Categories
- **PostgreSQL** - Relational, open-source, most extensible
- **MySQL** - Relational, huge in web apps (owned by Oracle)
- **Oracle** - Enterprise relational, expensive
- **Microsoft SQL Server** - Microsoft's relational offering
- **SQLite** - Embedded, no server, runs everywhere
- **MongoDB** - Document database, most popular NoSQL
- **Redis** - Key-value, in-memory, sub-millisecond reads
- **Apache Cassandra** - Wide-column, massive scale
- **Neo4j** - Graph database, most recognized
- **Elasticsearch** - Search engine, logging (ELK stack)
- **Snowflake** - Data warehouse, analytics
- **Pinecone / Weaviate / Qdrant / Chroma** - Vector databases (AI era)
