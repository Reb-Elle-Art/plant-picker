# Ron Character Sheet Template

A template for documenting and generating Ron identity cards — can be shared with others who want to create their own Ron identities.

---

## Workflow: Document First, Generate Second

**Step 1:** Document the identity specifics in this template format
**Step 2:** Present the documented specifics to the user
**Step 3:** Wait for user approval before generating images

This two-step approach cuts potential image generation costs — the user can review, adjust, and approve the specifics before any images are generated.

---

## Canvas Structure
- Landscape (16:9) single page
- Dark navy header bar at top, navy footer at bottom
- White content area between

## Header Section
- Left: Raccoon logo vector + "RON" in large clean font + small sparkle icons
- Center: Four adjectives describing this identity (e.g., "CURIOUS. HELPFUL. CLEVER. ADVENTUROUS.")
- Below header tab: "CARD X OF X — [IDENTITY_NAME] / [OUTFIT_NAME]"

## Main Renders (Top Center)
Three full-body CGI renders of the character:
- FRONT VIEW - standing pose with relevant prop
- 3/4 VIEW - slightly rotated
- SIDE VIEW - profile with prop/motion
- Height measurement annotation on right

## Expressions Column (Right Side)
- Navy header: "EXPRESSIONS"
- 4 vertical headshots showing different moods
- Each labeled with expression name (identity-specific)

## Bottom Grid (8 Panels)

### Panel 1: BUILD SCIENCE
- Species, Style, Head Ratio, Build, Ear Shape, Face Markings, Tail, Silhouette

### Panel 2: CORE PROPORTIONS
- Wireframe diagram with percentage labels
- Design logic bullet points with checkmarks

### Panel 3: COLOR PALETTE
- Color swatches with names and hex codes
- Identity-specific palette

### Panel 4: KEY ITEMS
- Isolated renders of props/accessories
- Alternative outfit thumbnails

### Panel 5: REBUILD RULES
- Design constraints
- Keep statements for consistency

### Panel 6: DYNAMIC POSES
- Character in environmental contexts
- Scene-specific poses

### Panel 7: CHARACTER BREAKDOWN
- Species, Role, World, Personality, Design Rules

### Panel 8: AT A GLANCE
- Descriptive phrases with personality-specific icon bullets

## Typography & Style
- Clean sans-serif typeface hierarchy
- Palette: Navy Blue (#001B3D), White, Light Grey, Dark Grey
- Professional, technical product schematic feel

## Footer
- "[IDENTITY] IP REFERENCE SHEET • CARD X OF X • [LOOK] • © HeyRon.AI"
- Small paw print icon

# Pluggable Stills Template

## Template

> Cinematic eye-level [SHOT_TYPE] of an anthropomorphic raccoon, [EXPRESSION], wearing [ATTIRE], [POSE] in [SETTING]. [BACKGROUND_DETAILS]. [LIGHTING], photorealistic 3D render, highly detailed fur, 8k resolution.

## Variables

| Variable | Description |
|----------|-------------|
| SHOT_TYPE | medium shot, medium-wide shot |
| EXPRESSION | Face/expression description |
| ATTIRE | Clothing from identity |
| POSE | Action/pose |
| SETTING | Environment type |
| BACKGROUND_DETAILS | Specific background elements |
| LIGHTING | Lighting mood |

## Filling the Template

1. **EXPRESSION**: Pull from identity's EXPRESSIONS column
2. **ATTIRE**: Pull from identity's KEY ITEMS
3. **POSE**: Pull from identity's DYNAMIC POSES
4. **SETTING/BACKGROUND**: Pull from personality traits
5. **LIGHTING**: Match to identity's COLOR PALETTE mood
